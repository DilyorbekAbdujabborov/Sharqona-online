✨ Features

  • 🔄 Real-time tracking of online users
  • 📍 User origin tracking via API
  • 📂 Optional static file serving

📡 API Endpoint

Track User Origin

Send a POST request to the following endpoint:

POST /api/track

Request Body:

{
    "origin": "<user-origin>"
}

Response:

{
    "message": "Origin tracked"
}

💬 WebSocket Connection

Connect to the WebSocket server at:

ws://localhost:3000

The server will notify you of the current number of online users! 🔔

✨ Using Simple Add the Script Tag

To add the users.js script dynamically, include the following in your HTML:

<script>
    const script = document.createElement('script');
    script.src = `http://${location.hostname}/users.js`;
    script.defer = true;
    document.head.appendChild(script);
</script>

This script will load users.js from the server based on the current hostname.
