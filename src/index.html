🌐 Welcome to the WebSocket Server Application!

Real-time user tracking and origin logging made easy! 🎉

✨ Features

  • 🔄 Real-time tracking of online users
  • 📍 User origin tracking via API
  • 📂 Optional static file serving

📡 API Endpoint

Track User Origin

Send a POST request to the following endpoint:

POST /api/track

Request Body:

{
    "origin": ""
}

Response:

{
    "message": "Origin tracked"
}

💬 WebSocket Connection

Connect to the WebSocket server at:

ws://localhost:3000

The server will notify you of the current number of online users! 🔔
